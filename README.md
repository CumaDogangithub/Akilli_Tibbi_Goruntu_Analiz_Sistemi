# 🩺 Akıllı Tıbbi Görüntü Analiz Sistemi (ATGAS)

Yapay zeka destekli tıbbi görüntü analiz platformudur. X-ray, MRI ve CT taramalarını analiz ederek potansiyel hastalıkları otomatik olarak tespit eder ve doktorlara hızlı, doğru teşhis koyma imkanı sunar.

## 🚀 Proje Özeti
Bu proje, tıbbi görüntüleme (DICOM, JPG, PNG) verilerini derin öğrenme modelleriyle analiz ederek sağlık profesyonellerinin teşhis sürecini hızlandırmayı ve insan kaynaklı hata payını en aza indirmeyi amaçlamaktadır. Görüntüler OpenCV ile ön işlemden geçirilir, TensorFlow/Keras tabanlı modellerle analiz edilir ve Flask ile geliştirilen kullanıcı dostu web arayüzü üzerinden doktorlara detaylı olarak raporlanır.

## 🛠️ Kullanılan Teknolojiler
* **Programlama Dili:** Python
* **Yapay Zeka & Derin Öğrenme:** TensorFlow, Keras
* **Görüntü İşleme:** OpenCV, pydicom
* **Web Arayüzü (Backend):** Flask

## 👥 Geliştirici Ekip
Bu proje, Agile/Scrum metodolojisi takip edilerek aşağıdaki ekip tarafından geliştirilmektedir:

* 👑 **Cuma Doğan** - Yazılım Mühendisi
* 💻 **Nihal Eylül İl** - Yazılım Mühendisi
* 💻 **Ozan Diyar Ay** - Yazılım Mühendisi
* 💻 **Esmanur Ulu** - Yazılım Mühendisi
